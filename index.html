About Tesla Joy

Tesla Joy is a student studying 2D/3D animation, modeling, an illustrater and a web designer.

Location: Austin, TX

Autobiography

When I really first got into animation, I was watching a bunch of cartoons show that aired on Television. I started to draw characters from those shows and then around in middle school I took animation class. Ever since that, I've decided to stick with animation and well soon 3D animation in high school. Technology is basically our future, my dad always tell me that having little knowledge into coding will be very helpful since it will mostly be used in a daily life style sooner or later. So I've taken coding courses as well as learning computer parts and using different software for digital art. I've always been around technology since I was young since my dad is a software engineer, so I have learned the language Python. I am currently learning HTML and CSS, which have been helping me create my alerts for my twitch streaming. I use animate, after effects and photoshop. I am learning Maya and Blender. Hopefully in the future I can go into Game Art to do Concept Art and/or 3D animation or the creation on putting it into the program and creating the animation for the character, including assets. Currently I am working on a portfolio website using my skills in HTML, CSS and JS. Then a big idea on creating a VR based game.
